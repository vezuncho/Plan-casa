# CasaPlan v60

Corrección completa de la imagen y los iconos de la aplicación.

Sube **el contenido descomprimido** de esta carpeta a la raíz del repositorio de GitHub. Deben verse en GitHub `index.html`, `manifest.webmanifest`, `version.json`, `_headers` y la carpeta `assets`.

Después del despliegue, elimina el acceso antiguo de la pantalla del móvil y vuelve a instalar CasaPlan desde la URL de Cloudflare para que Android regenere el icono.

No requiere cambios de SQL.
