# CasaPlan v43.1

## Cambios

- Administrador principal único.
- Panel de permisos independientes por miembro.
- El administrador principal puede transferir la administración.
- El administrador principal puede cambiar el PIN.
- Los miembros pueden recibir permisos concretos sin ser administradores totales.

## SQL

Ejecutar en Supabase:

`supabase/migracion_desde_v42_a_v43.sql`

No uses el SQL completo si ya tienes CasaPlan funcionando; usa solo la migración.
